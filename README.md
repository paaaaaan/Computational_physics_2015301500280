# 炮打翼龙

## 游戏说明
### 游戏思路借鉴于：[4399炮轰外星小怪](http://www.4399.com/flash/192577_2.htm)  额，这个要好玩多了，已玩通关
### 操作：可以通过w、a、s、d键控制大炮的位置，用鼠标调整射击角度，点击鼠标左键发射炮弹。上方有翼龙飞过，发射炮弹即可击中翼龙。
